Horizon Realty Advisors is looking for a Junior Web Developer with experience working on web applications in Laravel. You will work in an environment that puts you in direct contact with the end-users and values a results focused approach. You will also have an opportunity to be actively involved in the direction of the product, as well as work in an environment that values learning and constantly leveling up your skills.

Skills & Requirements:
• You have experience building web applications in PHP/Laravel.
• You have knowledge of and experience with the LAMP stack, Linux, modern PHP, git, MySQL, relational databases, REST, JSON, and related technologies.
• You have experience working with third-party APIs (Google, AWS, etc).
• You have development experience with HTML/CSS/JavaScript. Experience with modern JavaScript frameworks (React, Angular, etc.) is a plus but not required.
• Previous experience building RESTful APIs (design, maintenance, consumption) is a big plus.
• WordPress experience is also a big plus, especially working with the WP REST API.
• You have a project that you can show us and talk about. A portfolio and code samples (GitHub, etc) would be a big plus.
• You must be a strong, responsive communicator, be able to work independently, and have the ability to interpret the objective and help deliver a solution.
• You can take initiative to continuously improve the application, the development environment, and development processes, and can take responsibility for your work from day one.
• You have working proficiency and communication skills in both verbal and written English.

Please send the following information.
• CV/Resume
• Link to your portfolio
• Link to your GitHub repo
• Anything else you think will help us get to know you

Must pass a pre-employment / post offer background check.
